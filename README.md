# My-Portfolio
A modern, responsive single-page professional portfolio website featuring animated service cards, smooth scrolling, and a clean user interface. Built with HTML, CSS, and JavaScript.
Key Features:

Glassmorphism UI design.

Interactive and Animated Service boxes.

Smooth navigation to Home, Services, and Contact sections.

Fully Responsive (Mobile Friendly).

Technologies Used: HTML5, CSS3 (Custom Styles), Vanilla JavaScript.
